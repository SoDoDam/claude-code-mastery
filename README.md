# 개발자 웹 이력서 포트폴리오

현대적인 디자인의 개발자 웹 이력서 & 포트폴리오 사이트입니다.

## 📱 특징

- ✨ 반응형 디자인 (모바일, 태블릿, 데스크톱)
- 🎨 TailwindCSS를 활용한 현대적 스타일
- ⚡ 부드러운 스크롤 및 인터랙션
- 📝 이력서 및 포트폴리오 통합
- 📧 연락처 폼 기능

## 🛠️ 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3 + TailwindCSS** - 반응형 스타일
- **JavaScript (Vanilla)** - 인터랙션 기능

## 📂 프로젝트 구조

```
├── index.html          # 메인 페이지
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── main.js         # JavaScript 기능
├── assets/
│   └── images/         # 이미지 폴더
├── ROADMAP.md          # 프로젝트 로드맵
└── README.md           # 이 파일
```

## 🚀 시작하기

1. 파일을 다운로드하거나 클론합니다
2. `index.html`을 브라우저에서 엽니다
3. 필요에 따라 콘텐츠를 수정합니다

## 📝 콘텐츠 수정

`index.html`에서 다음 부분을 수정하세요:

- 이름/직급: `<h1>`, `<p>` 태그
- 자기소개: About 섹션
- 경력정보: Experience 섹션
- 프로젝트: Projects 섹션
- 기술스택: Skills 섹션
- 연락처: Contact 섹션
